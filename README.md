# dk-blog

A modern blog application built using the MERN stack principles on the frontend. Users can create, edit, publish, and manage blog posts with a clean and responsive UI.

---
## Features
- Create, edit, and delete blog posts
- Save blogs as draft or publish them
- Tag system (up to 5 tags per blog)
- Dynamic form handling using React Hook Form
- Context API for state management
- Persistent storage using localStorage
- Conditional rendering for empty states


---
## Tech Stack
- React
- React Router
- React Hook Form
- Context API
- Tailwind CSS
- React Toastify
